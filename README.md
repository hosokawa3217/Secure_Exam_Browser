# Secure_Exam_Browser

================================================================                
【Secure Exam Browser - システム仕様書（最新版）】          2026.02.10
================================================================
1. 概要:
   - 試験および特定業務への集中を目的とした、高セキュリティ型ブラウザ。
   - Chromium (PyQt6 WebEngine) を基盤とし、独自の外殻（ラッパー）で保護。

2. セキュリティ・制限機能:
   - 【起動制限】: 起動時に開いているすべてのフォルダ（エクスプローラー）を強制終了。
   - 【アプリ監視】: Chrome, Edge, Firefox, Brave, Discord の起動を常時スキャンし、検知時は画面をロック。
   - 【視覚ブロック】: 不正検知時、Webコンテンツを即座に隠蔽し、真っ赤な警告画面へ切り替え。
   - 【全画面固定】: タスクバーを完全に覆い隠すフルスクリーン表示（最前面固定）。
   - 【フォーカス監視】: ウィンドウが非アクティブ（Alt+TabやWinキー操作）になると、警告音を鳴らしロック。
   - 【操作制限】: マウスの右クリックメニューを完全に無効化（ソースコード閲覧やコピーの防止）。

3. 利便性向上機能（NEW）:
   - 【ログインID保存】: 外部ファイル (login_config.json) にログインIDを暗号化（プレーン）保存。
   - 【自動入力機能】: ページ読み込み完了時、JavaScriptを注入してログインIDを自動セット。

4. 特殊操作（管理者・デバッグ用）:
   - [Ctrl + Alt + L]: ログインID設定ダイアログの表示。
   - [Ctrl + Alt + S]: 強制進捗コマンド。警告状態をバイパスしてWeb表示を開始。
   - [Esc]: プログラムの終了。

5. 動作環境:
   - Python 3.10+, PyQt6, PyQt6-WebEngine, psutil
